# 🎨 UI Kit - Design System

## 📌 Descripción
Este proyecto es una biblioteca de componentes UI reutilizables desarrollada con HTML y CSS puro.

## 🧩 Componentes incluidos
- Botones (primary, secondary, danger)
- Formularios (inputs y textarea)
- Cards para tickets
- Alertas de estado
- Badges de prioridad

## 🚀 Tecnologías
- HTML5
- CSS3 (variables, flexbox básico, hover effects)
